  • git clone path-to-repository
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить новий файл под контроль Git
  • git commit -am "added new file index/html" - зафиксировать изменения и дать и коментарий
  • git push - отправить все закомичение изменения в репозиторий на сервер
  • git pull - забрать свежие обновления
  • gitk --all& - открыть графическое окно с ветками
